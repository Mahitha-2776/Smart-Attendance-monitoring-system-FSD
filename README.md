## 📌 About the Project

Smart Attendance Monitoring System is a responsive frontend web application developed as part of the Full Stack Development (FSD) Lab Project.

It simulates a real-world attendance management system with role-based access for **students** and **admins**, featuring attendance tracking, visual analytics, and department-wise reporting — all built using HTML, CSS, and Vanilla JavaScript.

---

## 🚀 Features

- 🔐 Role-based Login — **Student** and **Admin** roles
- 📊 Admin Dashboard to manage and monitor attendance
- 🎓 Student Dashboard to view personal attendance per subject
- 📈 Visual Analytics — Heatmap, Pie Chart, Weekly Trends
- 🏫 Multi-department support — CSE, ECE, ME, CE
- 📋 Section-wise attendance records
- ⚠️ Attendance warning insights panel
- 🌙 Dark/Light theme toggle
